Class: connection

connection(host)

new connection(host)

The connection class is used to handle the communication with the Velux KLF interface. It provides login and logout functionality and provides methods to run other commands on the REST API.
Parameters:
Name Type Description
host string URL of the host name, e.g. http://velux-klf-12ab
Source:
  • connection.js, line 29
Example
const connection = require('velux-api').connection;

let conn = new connection('http://velux-klf-12ab');
conn.loginAsync('velux123')
    .then(() => {
        ... do some other stuff ...
        return conn.logoutAsync();
     })
    .catch((err) => {    // always close the connection
        return conn.logoutAsync().reject(err);
     });
